05Agentic, not automaticThe agent gathers and suggests. Humans decide.
06Open methodPortable across stacks. No vendor lock.

Statement

AURA is a simple agentic documentation method for project and product work. It connects the signals your team already produces — code changes, pull requests, issues, sprint activity, KPIs — and turns them into a structured weekly review.

Not a heavyweight framework. Not another dashboard. Not static documentation that goes stale.

It replaces status theater with operating visibility.

Questions AURA answers every cycle

→ What changed?
→ What shipped?
→ What is off track?
→ What do metrics say?
→ What needs attention next?

§ 05.1 — CONSTRUCTION

Drawn on a 24 × 24 grid with 2px keyline padding.

Strokes align to the pixel grid. Corners are square unless rounding is semantic.
Construction specification
Canvas24 × 24 pixels
Keyline20 × 20 inner, 2px padding
Stroke1.25px, uniform
Corner radius0 default · 2 semantic
Line capsround
Line joinsround
Angles15° increments, prefer 45°/90°
Min size16px display · 12px inline
File nameaura/icon-[name]-24.svg
Metaphorliteral, not decorative
Rules of thumb
  1. One metaphor per icon. Never stack concepts.
  2. Prefer circles to ovals. Squares to rounded rectangles.
  3. When in doubt, remove a line.
  4. The core dot is reserved for signal / source / focus.

§ 06 — VOICE

Plain. Precise. Grounded in evidence.

If a sentence doesn't carry information, cut it.
Specific
Name the PR, cite the metric, link the ticket. Avoid phrases like "a lot of good progress."
Short
Short sentences. Active verbs. One idea per line. A weekly brief is read in minutes, not hours.
Honest
If a sprint slipped, say so. If a KPI moved the wrong way, name the direction and the suspected cause.
Calm
Never use hype, urgency icons, or alarm language unless the alarm is real and actionable.
Lowercase-leaning
Titles and UI can sit lowercase. Proper nouns stay cased. ALL CAPS is for labels only.
Owner-aware
Every recommendation ends with a name or a role. "Needs attention" is not a directive.
Say this / not that
Do
Cycle time fell 22% after PR-4817 shipped. Watch for regression next sprint.
Don't
Team crushed it this week! Huge wins across the board 🚀
Do
Blocker: auth migration pending security review. Owner: @mira.
Don't
There are some things slowing us down, but nothing we can't handle.

§ 08 — CLI

One prompt character. One accent color. No emoji.

AURA terminal output is styled like a linter — concise, status-forward, link-rich.
