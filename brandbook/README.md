# AURA Brand Assets

Visual identity for OpenAURA — not shipped with the Python package. This directory
exists so contributors have the logo, colors, and slide templates in one place.

## Contents

- **AURA Brandbook.html** — interactive brand book: logos, palette, typography, voice,
  imagery. Open in a browser.
- **deck-stage.js** — animated slide / stage component used in the brandbook.

## Usage

These assets are for README graphics, presentations, and marketing. The runtime
package (`openaura/`) does not load anything from this directory.
